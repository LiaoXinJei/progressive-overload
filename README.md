# RP LOG V3.0 - Progressive Overload Tracker

一個基於 Renaissance Periodization 原則的訓練追蹤應用程式，幫助你科學化地管理訓練量與恢復。

## 功能特色

- 📊 **週期化訓練**：10 週漸進式訓練計劃
- 💪 **雙模式切換**：
  - 維持模式（腿部專項）
  - 增重模式（全身發展）
- 📈 **訓練量視覺化**：即時追蹤各肌群訓練組數
- ✅ **組數追蹤**：點擊記錄完成的訓練組
- 💾 **本地儲存**：訓練進度自動儲存於瀏覽器

## 開始使用

### 安裝依賴

```bash
npm install
```

### 本地開發

```bash
npm run dev
```

### 建置專案

```bash
npm run build
```

### 預覽建置結果

```bash
npm run preview
```

## 訓練哲學

基於 RP（Renaissance Periodization）的訓練量管理原則：
- **MV（Maintenance Volume）**：維持肌肉量的最低訓練量
- **MEV（Minimum Effective Volume）**：開始產生增長的訓練量
- **MAV（Maximum Adaptive Volume）**：最佳增長的訓練量
- **MRV（Maximum Recoverable Volume）**：最大可恢復訓練量
